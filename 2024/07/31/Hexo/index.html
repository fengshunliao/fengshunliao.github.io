Git and Nodejs

requirement installations:

Git

  • open gitbash
  • $ ssh-keygen -t rsa -C "your_email@example.com"
  • press “enter” until you get public key ( will show the saving path in terminal )
  • copy the code in “id_rsa.pub”
  • add to github setting
  • test connection in gitbash $ ssh -T git@github.com

Node.js

Install Hexo

  • open windows powershell
  • npm install hexo-cli -g
  • mkdir your_hexo_dir, cd your_hexo_dir

Init Hexo

  • hexo init

Run Hexo in local

  • hexo generate generate static files
  • hexo server run hexo in local

Run Hexo as github page

  • repository name like githubname.github.io
  • in hexo project find _config.yml
  • run hexo generate and hexo deploy
  • you will be required to login github
  • if see ERROR Deployer not found: git run npm install hexo-deployer-git --save at “your_hexo_project_name” route
  • run hexo deploy again
  • set github pages and visit site:

Hexo Theme Butterfly

https://butterfly.js.org/posts/21cfbf15/


install with npm

  • npm install hexo-theme-butterfly
  • you can update theme with npm update hexo-theme-butterfly
  • open _config change theme name
  • download renderer npm install hexo-renderer-pug hexo-renderer-stylus --save
  • generate and deploy hexo generate, hexo deploy
