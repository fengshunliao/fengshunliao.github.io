Install

Open PowerShell and enter the command wsl --install, then restart computer.
After that your are required to set up your linux user info, input username and password.

Update and upgrade packages

Run sudo apt update or sudo apt upgrade .

GUI for Linux

Prerequisites: respective GPU driver of your computer.

  • Update wsl and restart: wsl --update, wsl --shutdown.
  • Open wsl and update packages of your current distribution: sudo apt update.
  • Install Gnome Text Editor: sudo apt install gnome-text-editor -y
  • Install GIMP graphic editor: sudo apt install gimp -y, to launch gimp
  • Install Nautilus file manager: sudo apt install nautilus -y, to launch nautilus
  • Install VLC multimedia player: sudo apt install vlc -y, to launch vlc

Install browser (Chrome)

  • cd /tmp
  • wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
  • sudo apt install --fix-missing ./google-chrome-stable_current_amd64.deb
  • to launch google-chrome

Install Microsoft Edge:

  • check https://www.microsoft.com/zh-cn/edge/download/insider?platform=linux-deb

File Storage

In Windows, directly open file explorer.
In Linux, enter command explorer.exe to open same path.

Overview Practices

  • Set up VS Code with remote development.
  • Set up version management with Git.
  • Set up remote development containers with Docker.
  • Set up a database.
  • Set up GPU accelerated machine learning traning in WSL.
    Overall using wsl as a remote linux development environment, run docker or databases to help development.

Basic commands

  • wsl --install,wsl --update,wsl --shutdown
  • wsl ls -la to list files in the directory.
  • wsl ls -la | findstr "git" to filter the results for words contain “git”.
  • notepad.exe .bashrc to open .bashrc file with notepad on windows machine

Install VScode

  • Download and install Vscode on Windows. Be sure Add to PATH when install.
  • Open vscode and install Remote Development extension pack(search extension).
  • On Linux allow SSL sudo apt-get install wget ca-certificates
  • Install Vscode server on Linux : code . vscode server then run on Linux

Using Windows Vscode to connect Linux Vscode:

  • Click View -> Command Palette, input “WSL”
  • Click Connect to WSL
  • When running the WSL extension, selecting the Extension tab will display a list of extension split between your local machine and your WSL distribution.

The image below shows the connection to the remote Linux, click it you can chose to connect or close the remote.

Install Git

Git can be installed on both Windows and WSL.
Download and install on Windows: https://git-scm.com/
Download and install on Linux: sudo apt-get install git

Git config file setup on both Win and Linux ( To tell the repository who uploaded files ):

  • git config --global user.name "Your Name"
  • git config --global user.email "youremail@domain.com"
  • nano ~/.gitconfig to editor git config

Key RSA:

  • Windows: ssh-keygen -t rsa -C "youremail@domain.com" step by step with default setting.
  • Message: “Your public key has been saved in /c/Users/tianx/.ssh/id_rsa.pub”
  • Copy the content in id_rsa.pub, login github, select “setting>SSH and GPG keys>New SSH key”
  • Test if successful, ssh -T git@github.com
  • Same on Linux

Credential Manager setup?

  • https://github.com/microsoft/Git-Credential-Manager-for-Windows download and install
  • git --version check git version on Linux, if <v2.36.1 install using git config --global credential.helper "/mnt/c/Program\ Files/Git/mingw64/bin/git-credential-manager-core.exe"

Install Databases

Choices:

  • MySQL
  • PostgreSQL
  • Microsoft SQL Server
  • SQLite
  • MongoDB - NoSQL
  • Redis - NoSQL

Install MySQL on Linux:

  • sudo apt install mysql-server
  • mysql --version
  • systemctl status mysql start mysql
  • sudo mysql open prompt
  • since the prompt is opened, you can do:
  • SHOW DATABASES; this is a sample command
  • ALTER USER 'root'@'localhost' IDENTIFIED BY '_new_password_';replace the _new_password_ with your own password

Connect from Windows:

  • Download heidisql: https://www.heidisql.com/download.php and install.
  • Ensure to run mysql server, with sudo systemctl status mysql
  • open cd /etc/mysql/, sudo nano my.cnf modify as follows:
  • Restart mysql server sudo service mysql restart
  • To create a remote user account: sudo mysql -u root -p enter password
  • Then do as follows:
  • Open HeidiSQL and connnect to WSL sql server

Official documents: https://dev.mysql.com/doc/mysql-getting-started/en/
